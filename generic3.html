VARIÁVEIS

Variáveis são "caixas" que tem um nome e podem armazenar informações ou dados, como um nome ou o valor de um produto. Elas podem ter diversos tipos que podem mudar durante a execução, sendo eles:
int = número inteiro
float = número decimal
str = palavra, cercada de '' ou "", tanto faz
bool = valor verdadeiro ou falso (True ou False)
list = conjunto de outros valores entre [] e separados por ,
Existem outros tipos de dado ou variável que nós vamos falar mais à frente

A seguir temos um exemplo do uso desse conceito: 

nome = 'Aquino'   # str
idade = 16        # int
altura = 1.72     # float
estudante = True  # bool
alunos = ["Miguel", 'Theo', "Aquino"] # list

print('Oi, ' + nome + "!")

OPERADORES

Os operadores são caracteres que alteram os valores de outras variáveis ou dados. A maioria desses operadores só atuam em variáveis com valores numéricos como ints e floats mas alguns também funcionam com str, de uma forma diferente dos valores numéricos.

O primeiro operador é o "=", que nós já usamos anteriormente e que não é o sinal de igual mas sim de assinar, ou seja, esse operador vai assinar um valor para outro da direita para a esquerda:
variavel = 1 (1 passa para a direita e a var vira 1)

Operadores para ints e floats: 

n = 2 + 3  # 5  | operador de adição            + 
n = 3 - 2  # 1  | operador de subtração         - 
n = 3 * 2  # 6  | operador de multiplicação     *   (x) 
n = 4 ** 2 # 16 | operador de potenciação       **  (^) 
n = 8 / 2  # 4  | operador de divisão           /  
n = 7 // 3 # 2  | operador de divisão sem resto // 
n = 5 % 2  # 1  | módulo / resto da divisão     %

Operadores para strings: 

nome = "Aqui" + "no"        # 'Aquino' | concatenação     + 
fruta = "Ba" + ("na"*2)     # 'Banana' | multiplicação    * 
temA = 'A' in "Lorem Ipsum" # False | "tem X em Y?"   in 
naoTemIp = 'Ip' not in "Lorem Ipsum" # False |    not in # not in  = contrário do in

Encurtamento de operações:

O operador = pode encurtar outras operações quando combinado com outros 

n += 1   # n = n + 1 
n -= 2   # n = n - 2 
n *= 10  # n = n * 10 
n /= 2   # n = n / 2

LOOPS

As estruturas de repetição ou loops são muito importantes para que possamos repetir um bloco de código várias vezes. Existem dois loops no Python: o while e o for in.

O while loop repete algo até certa condição ser falsa, um número muitas vees indeterminado de vezess, e consequentemente ele deve modificar essa condição pois caso contrário esse loop será infinito. Por exemplo: 

repeticoes = 4
while (repeticoes > 0):
  print('Olá ', repeticoes)
  repeticoes -= 1;           # Atualizando o numero de repeticoes
# "Olá 4", "Olá 3", "Olá 2", "Olá 1"

Já o for in loop vai se repetir uma quantidade certa de vezes passando por cada elemento de uma lista ou str por exemplo. Por exemplo: 

palavra = 'banana'
palavra_B = ''
participantes = ['Miguel', 'Theo', 'Aquino']
contagem_pessoas = 0

INPUTS

Inputs são as entradas ou valores que o usuário coloca no programa e que podem ser usadas. Eles podem ser obtidos por meio do comando input(frase_de_comando)

Exemplo de uso: 

nome = input('Insira seu nome: \n') # *
print("Olá, " + nome + "!")
# Insira seu nome:
# Aquino 
# Olá, Aquino!
# * O comando \n gera uma nova linha no programa

COMENTÁRIOS

Indo em frente, os comentários são partes do programa que são ignoradas na execução e que servem para guiar o programador e explicar o que o código está fazendo.
Para criá-los, basta digitar # e depois desse caractere, todo o resto da linha será transformada em um comentário

CONDICIONAIS

As operações condicionais tornam o nosso programa dinâmico, permitindo que nós possamos mudar o rumo do programa com base em certas condições, como por exemplo se a senha e login estão corretos.

Antes de aprendermos as estruturas condicionais, é importante entender os operadores condicionais (igual a, diferente de) que vão comparar dois valores e retornar True (verdadeiro) ou False (falso) com base na sua comparação. Esses operadores são:

x = 4
y = 5
z = 4
x2 = x
x == y      # False | operador de igualdade
x != y      # True  | operador de desigualdade
x is z      # False | operador de igualdade absoluta *
x is not x2 # False | operador de desigualdade absoluta *
x > y       # False | maior que
x < y       # True  | menor que
x >= z      # True  | maior ou igual a
x <= y      # True  | menor ou igual a

A igualdade absoluta significa que as duas variáveis são o mesmo objeto/variável, e retorna False mesmo se as duas variáveis tiverem o mesmo valor

Estruturas condicionais:
As estruturas condicionais usam esses valores de True or False para determinar e mudar o rumo do programa, e elas são bem úteis ao se tomar decisões. Os If/Elif/Else funcionam da seguinte forma:

estoque = 11
if (estoque <= 5):   # Condição a ser testada
    print('Estoque quase vazio') # Código executado se a condição for verdadeira
elif (estoque == 0): # Outra condição a ser testada se a o if for falso
    print('Estoque vazio') # Se True, executa isso
... # * podem ser usados infinitos elif
else:                # Se todas forem falsas, executar isso:
    print('Estoque normal')
